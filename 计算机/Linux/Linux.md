# Linux

## 一、文件系统

### 1.1 文件分隔符

Linux下文件分隔符为`/`

Windows下为`\`



## 二、Vim编辑器

### 2.1 一般模式

`y$`复制从光标位置到行末的字符，按`p`进行粘贴

`y^`赋值从光标位置到行开始的字符，按`p`进行粘贴

### 2.2 命令模式

`Shitf + o`也就是`O`，可以在光标上面追加一列空行并将光标移动至改行开始处

`:wq!`强制修改没有写权限的文件9854



## 三、界面切换

`Ctrl + Alt + F(12345...)`切换图形界面和命令界面

也可以在图形界面输入命令`init 3`切换到命令界面（运行级别3）

在命令界面输入命令`init 5`切换到图形界面（运行级别5）
